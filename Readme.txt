It's a template for 3D games or CG Project.
Default scene "Test Scene" is an example for all things this engine have and also a manual that describe how to use.

To compile, 
1. Use CMake, and configure with "CMakeLists.txt". 
	Since this program is tested on 
		windows10, 
		Visual studio 2019, 
		x64 Platform,
		it is recommended to use the same setting. Then generate project file.
	
2. Open LittleEngine.sln, compile LittleEngine with 
	Solution Configuration: Release
	Solution Platform: x64
